## Descripción

Base de datos que compila información sobre empleo en Latinoamérica por país, con evolución temporal cada 5 años desde el año 2000 hasta 2020. Incluye tasas de empleo, participación laboral y otros indicadores del mercado laboral para múltiples países latinoamericanos.
